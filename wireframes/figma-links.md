This Figma file contains the low-fidelity mobile wireframes and clickable prototype for the C&C Feed Inventory app.

The prototype includes:

- Login
- Home
- Take Feed
- Customer flow
- K2 flow
- Family flow
- Add Stock
- Inventory
- Invoices
- Accounts
- Activity History
- Reports
- Profile / Settings
- Empty states
- Error / warning states

## Important Reminder

The Figma Make code is a prototype foundation, not final production logic.

The final app still needs deliberate development for:

- database schema
- authentication
- roles and permissions
- real inventory transactions
- invoice/payment logic
- activity history
- CSV import
- reporting
